TLDR - stat
==========

Overview
--------

[stat] displays the status of the file or file system, as specified.  It includes date last edited, size of file, and file ID, among other things.
Examples
--------

- Display the **status** of a specified *file* sqrt.c.

        $ stat sqrt.c
